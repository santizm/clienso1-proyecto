Acerca del Proyecto ClienSO

ClienSO es una empresa ficticia utilizada como caso de estudio para implementar servicios de red en un entorno Windows Server.

El proyecto tiene como objetivo:

  • Implementar autenticación centralizada mediante Active Directory.
  • Configurar servicios de DNS, DHCP y FSRM.
  • Desplegar un servidor web IIS con HTTPS y certificado interno.
  • Aplicar políticas de grupo y redirección de carpetas.

Este sitio web es parte de la documentación y demostración de funcionamiento del laboratorio.
