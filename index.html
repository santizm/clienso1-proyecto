Bienvenido a ClienSO

Este es el sitio web básico del proyecto de laboratorio de Sistemas Operativos 1. Aquí se centraliza la información sobre los servicios implementados: Active Directory, DNS, DHCP, FSRM, DFS y más.

El objetivo de esta demostración es mostrar la capacidad de integrar y administrar servicios en un entorno Windows Server.
