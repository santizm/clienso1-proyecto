Servicios Implementados

Estos son los servicios instalados y configurados en el entorno del proyecto ClienSO.

Active Directory

Autenticación centralizada, control de usuarios y grupos, políticas de seguridad y administración de dominio.

DNS

Resolución de nombres interna para el dominio clienso.local y los recursos compartidos.

DHCP

Asignación automática de direcciones IP en la red interna 192.168.10.0/24 configurada por PowerShell.

FSRM

Gestión de cuotas de almacenamiento y restricción de tipos de archivos (MP3, AVI, etc.).

IIS y Web Server

Servidor web con sitio www.clienso.local accesible por HTTP y HTTPS con certificado emitido por la CA interna.

Políticas de Grupo

Configuración automática de escritorio, bloqueo de panel de control, redirección de carpetas y mapeo DFS.
